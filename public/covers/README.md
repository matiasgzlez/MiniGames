# Portadas de los juegos

La landing busca la portada de cada juego en `/covers/<id>.jpg` (este directorio).
La card muestra la imagen sola (el nombre va DENTRO de la imagen); si el archivo
no existe, queda un fallback con el nombre del juego.

- Formato: JPG (o renombrar a .jpg), cuadrado **1:1**, ideal 1024x1024.
- El CSS aplica los bordes redondeados: generar la imagen con esquinas rectas.
- Cada bloque ya incluye estilo base + escena + titulo dentro de la imagen:
  **copiar y pegar tal cual** en el generador de imagenes.
- Si el texto del titulo sale deformado, regenerar; los modelos a veces
  necesitan 2-3 intentos para tipografia limpia.

franja horizontal del medio. Claves: escena oscura, sin texto, y los elementos
importantes en la banda central (arriba y abajo se pierden con el recorte).
La pagina le pone un velo oscuro a la izquierda para el texto, asi que el lado
derecho puede ser mas brillante.
